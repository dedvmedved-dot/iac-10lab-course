# Приложение Г — Верификация GFS2 + iSCSI

**Статус**: 🟡 Требует проверки
**Дата**: 2026-06-25

## Ключевой результат

Приложение Г — продвинутая тема shared storage. Отличается от остальных лабораторных: нет массового создания ВМ через Terraform. Нет inventory с жёсткой привязкой к `student`. Основные сквозные баги не применимы.

Требует полной верификации с развёртыванием на отдельном стенде.
